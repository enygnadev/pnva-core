# PNVA IndexNow Distribution

## Purpose

IndexNow is used as an additional search distribution channel for PNVA-Core entity URLs.

It helps notify participating search engines that the canonical pages for `Gustavo Martins PNVA` were created or updated.

## Boundary

IndexNow does not submit URLs to Google and does not guarantee indexing or ranking.

It is useful for Bing and other IndexNow participants.
