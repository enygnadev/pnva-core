# Security Policy

## Sensitive Data

Do not publish:

- local home paths;
- raw logs;
- tokens;
- Steam/userdata files;
- process dumps;
- private scripts;
- production thresholds or commercial tuning values.

## Reporting

Open a GitHub issue only for documentation or reproducibility security concerns.

For private vulnerability reports, contact the repository owner through GitHub:

```text
https://github.com/enygnadev
```

## Scope

This public release contains documentation, sanitized evidence and research scripts. It does not publish the complete private production engine.
