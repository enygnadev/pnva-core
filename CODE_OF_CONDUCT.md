# Code of Conduct

PNVA-Core discussions should remain technical, evidence-based and respectful.

Expected:

- cite evidence;
- separate hypothesis from proof;
- disclose limitations;
- avoid personal attacks;
- keep security and privacy in mind.

Not accepted:

- harassment;
- spam;
- unsupported miracle claims;
- publishing personal data or raw private logs;
- presenting PNVA as medical, legal or physical certainty.
