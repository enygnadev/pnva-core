# Contributing

PNVA-Core is currently an open research / production evidence release.

## What Contributions Are Welcome

- Independent reproduction attempts.
- Review of the equations and validation protocol.
- Issues about unclear documentation.
- Suggestions for better benchmarks.
- Sanitized proof artifacts from other machines.

## What Is Not In Scope Yet

- Claims that PNVA is a universal physical theory.
- Medical, legal or safety-critical automation without licensed experts.
- Requests for the private commercial tuning layer.
- Unsanitized logs containing personal paths, tokens or user data.

## Reproduction Report Template

Open an issue with:

```text
hardware:
OS:
kernel:
desktop/session:
PNVA version:
gate duration:
result:
artifact:
notes:
```

Attach only sanitized JSON.
